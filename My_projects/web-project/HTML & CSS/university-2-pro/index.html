فورمه درخواستی شخص انفرادی

فورمه درخواستی شخص انفرادی به منظور ثبت ويا(تجديد) يک شخص انفرادی وهرنهاد انفرادی که اين شخص يگانه مالک آن می باشد،استفاده ميگردد. ماليه دهنده انفرادی يک شخصيت حقيقی ميباشد (نه نهاد حقوقی) بخش -١ " دليل تکميل فورمه " را جهت خانه پری ضمايم وفورمه های مناسب به دقت مطالعه نمائيد.
